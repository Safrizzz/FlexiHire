# FlexiHire

FlexiHire is a role-based job matching mobile application designed to connect **students** with **employers** for flexible and part-time job opportunities. The system provides an integrated platform for job discovery, job posting management, application tracking, messaging, and **instant payment upon job completion**, which distinguishes FlexiHire from conventional job platforms.

This project is developed as an academic software engineering project and demonstrates the application of mobile development, system analysis, and role-based navigation concepts.

## ğŸ‘¥ User Roles

The system supports three main user roles:

### 1. Administrator (Internal Operational Stakeholder)
- Manage user accounts
- Monitor job postings
- Oversee payment records
- Maintain system integrity

### 2. Student (External Stakeholder)
- Browse and search job listings
- Apply for jobs
- Track job and payment status
- Receive **instant payment upon job completion**
- Communicate with employers
- Manage personal profile

### 3. Employer (External Stakeholder)
- Create, edit, and delete job postings
- View and manage applicants
- Confirm job completion
- Trigger **instant payment release**
- Communicate with students
- Manage company profile
